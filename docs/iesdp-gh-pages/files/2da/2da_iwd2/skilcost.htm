skilcost.2da

This file set the bonus at skills for each class/kit/race/subrace, as other minor things.
The first column is the skill system name, the second its ID.
NAME_REF and DES_CREF each set string reference in dialog.tlk (for the name of the skill and its description).
The "UNTRAINED" column dictates if it is possible to use this skill if your character has no points in it).
