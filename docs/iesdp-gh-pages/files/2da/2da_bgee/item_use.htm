item_use.2da

This file defines how specific items and actors interact with each other.

First column defines the item.
Second column defines the actor's script name.
Third column defines the string that is displayed when item/actor combination matches the behaviour defined in the fourth column.
Fourth column is a bitfield that defines the item/actor behaviour:
  • Bit 0: When attempting to remove item from actor.
  • Bit 1: When another actor attempts to equip the item.
  • Bit 2: When attempting to remove item from actor without replacing it by another item.
