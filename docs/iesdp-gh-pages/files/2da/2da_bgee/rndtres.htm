rndtres.2da

This file defines what items are generated when a creature uses an item of filename defined in the first column. The dummy item is replaced by the generated item. Therefore, this works like a random treasure table.

The first column of RNDTRES.2DA specifies the name of the row, which is also used as the dummy item name which is replaced by the actual item. The rest of the columns have headers 1, 2, 3, 4, 5, ... They can be used to add the item resource which is replaced by the dummy item defined in the first column. The chance to get an item is calculated by dividing it by the number of defined columns (excluding columns consisting of '*'). The number of supported columns is open ended. You can add more columns if you need finer control over the percentages for each item.
