npcnd.2da
This file defines the portrait and text that will be displayed for each NPC during the epilogue sequence.

The default value field (in the sample file this is set to *NJAHEIRL) specifies the BMP file to use as the epilogue portrait.
The first column contains symbolic identifiers for particular classes and races. The second and third columns contain string references to indicate the epilogue text to display.
