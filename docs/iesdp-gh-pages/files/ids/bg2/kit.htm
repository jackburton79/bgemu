BG2: ToB Identifiers

Relates to offset 0x0244 in the .cre file format.
Used by the Kit(O:Object*,I:Kit*KIT) trigger, and the AddKit(I:Kit*KIT) and AddSuperKit(I:Kit*KIT) actions.
The file is extensible, and new entries can be added.
Note that the original file provided with the game is incorrect, and contains incorrect values. The updated version presented on tihs page is confirmed to work (though for optimal usage, certain kitted NPC's should have their kit values altered to match this file). The original (incorrect) kit.ids file is available here .
